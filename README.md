# PhyrasFury

Unity 2D spaceshooter project (Goblin Thrower).

## Requirements

- **Unity:** 2019.4.40f1 (see `ProjectSettings/ProjectVersion.txt`)

## Getting started

1. Clone the repository.
2. Open the project folder in Unity Hub and launch with the editor version above.
3. Open the main scene from `Assets` and press Play.
